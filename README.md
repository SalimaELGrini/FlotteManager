# 🚗 Fleet Management System

A modern web application developed using Laravel to help companies manage their fleet of vehicles efficiently. The system centralizes information about vehicles, drivers, maintenance, and daily operations through an intuitive dashboard.

## 📖 Project Overview

Managing a fleet manually can be time-consuming and prone to errors. This application was created to simplify fleet management by providing an organized platform where administrators can manage vehicles, monitor information, and perform daily operations efficiently.

## ✨ Features

- Secure user authentication
- Dashboard with fleet overview
- Vehicle management
- Add, edit and delete vehicles
- Search and filter vehicles
- Form validation
- Responsive user interface
- Database management with MySQL

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/SalimaELGrini/Gestionnaire-de-flotte.git
```

2. Install dependencies

```bash
composer install
```

3. Copy the environment file

```bash
cp .env.example .env
```

4. Generate the application key

```bash
php artisan key:generate
```

5. Configure the database in the `.env` file.

6. Run migrations

```bash
php artisan migrate
```

7. Start the development server

```bash
php artisan serve
```
